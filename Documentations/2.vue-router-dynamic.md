## ðŸ§  What Is a Dynamic Route?

A **dynamic route** captures part of the URL using a `:` prefix.

Example path:

```ts
"/projects/:id";
```

This allows URLs like:

- `/projects/1`
- `/projects/my-awesome-app`
- `/projects/vue-router-guide`

```ts
}
```
